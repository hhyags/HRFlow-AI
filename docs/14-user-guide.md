# HRFlow AI User Guide

## Access

1. Open the invitation link or enter the invitation code during signup.
2. Create an account with work email/password or Google.
3. Verify the email address before entering the dashboard.
4. Use Forgot Password when credentials need to be reset.

## Employee

- Review personal attendance and request corrections.
- Submit leave requests and track approval status.
- Review leave balances and payroll records.
- Open signed document links only while they are valid.

## Recruiter

- Create and manage open jobs.
- Upload candidate resumes.
- Review resume analysis, match score, and skill gaps.
- Generate interview questions and update candidate stages.

## HR Manager

- Manage employees, attendance, leave, payroll, documents, and organizational settings.
- Invite Recruiters and Employees.
- Use HR Copilot and analytics for reports and workforce insights.
- Review AI recommendations as decision support; retain human review for employment decisions.
